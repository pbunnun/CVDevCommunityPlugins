# CVDevCommunityPlugins

This repository contains community plugins for [CVDev](https://github.com/pbunnun/SeeWeDev), distributed under their own licenses.

## Plugins

- **GPLNodes**: MQTT Publisher and Subscriber nodes, and related widgets, for integrating MQTT messaging into CVDev workflows. These nodes use GPL-licensed dependencies (e.g., QtMqtt) and are distributed under GPLv3.

## Usage

To use these plugins in CVDev:
1. Add this repository as a submodule:
   ```sh
   git submodule add https://github.com/pbunnun/CVDevCommunityPlugins.git CVDevCommunityPlugins 
   ```
2. Build CVDev with the submodule included.
